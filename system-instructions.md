Create a premium PWA for "Craigs-Catch" - a Craigslist monitoring application that tracks items in real-time and sends notifications via Telegram bot. The app must have a sophisticated, premium aesthetic that replaces the dated Craigslist-style design.

TECHNICAL REQUIREMENTS:
- Build as a modern PWA with Next.js, TypeScript, Tailwind CSS
- Implement real-time Craigslist RSS feed monitoring
- Integrate Telegram bot notifications API
- Include filter management dashboard
- Responsive design for mobile/tablet/desktop
- Offline capability with service workers
- Push notification support

DESIGN SPECIFICATIONS:
- Color palette: Deep navy (#0F172A), premium gold (#D4AF37), soft white (#FAFAFA), charcoal (#334155)
- Typography: Inter for body text, Space Grotesk for headings
- Layout: Clean, spacious design with generous whitespace
- Cards: Subtle shadows (0 4px 6px -1px rgba(0,0,0,0.1)), rounded corners (lg)
- Buttons: Gradient primary buttons (gold to amber), outlined secondary buttons
- Navigation: Minimal top navigation with logo and settings
- Data tables: Modern striped rows with hover states
- Filters panel: Collapsible, premium card design with icons

USER INTERFACE ELEMENTS:
- Hero dashboard with notification statistics
- Real-time item feed with card layout showing item images, prices, locations
- Filter management panel with location, category, price range selectors
- Telegram connection status indicator
- Search functionality with instant results
- Notification history log
- Settings page with notification preferences

INTERACTIVE FEATURES:
- Live updating item cards with fade-in animations
- Toggle switches for enabling/disabling categories
- Drag-and-drop filter reordering
- Hover effects on all interactive elements
- Smooth transitions between views
- Loading skeletons during data fetches
- Toast notifications for user actions

BRAND IDENTITY:
- Logo: Modern "CC" monogram with subtle gold accent
- Tagline: "Smart Craigslist Monitoring"
- Premium feel through elevated components and refined spacing
- Professional typography hierarchy
- Sophisticated iconography (Lucide React)

FUNCTIONALITY WIREFRAME:
1. Dashboard: Stats overview, recent catches, quick filters
2. Feed: Real-time item cards with Telegram notification trigger
3. Filters: Advanced Craigslist category/location management
4. Notifications: History and Telegram bot settings
5. Settings: Account preferences and integrations

The final design should feel like a premium SaaS product, not a Craigslist clone. Focus on elegant typography, sophisticated color usage, and professional UI patterns that convey trust and reliability. Prioritize user experience for power users who monitor multiple Craigslist feeds simultaneously.
